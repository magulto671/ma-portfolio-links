## About This Project

This project was created as a beginner-friendly personal portfolio page while I continue learning web development. It includes a hero section, contact links, project cards, and an about section.

The goal of this project was to practice building a clean, organized webpage using HTML and CSS while creating something I can share on GitHub and LinkedIn.
